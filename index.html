Classe de Ação - Actions

Os métodos que a classe Actions fornece para a ação de arrastar e soltar:

    1-dragAndDrop (WebElementsource, WebElement target)
    2-dragAndDropBy (WebElementsource, int xOffset, int yOffset)

Ação de Duplo click:

    actions.contextClick (webElement).perform();

Os métodos que a classe Actions forneceu para a ação Mouse Hover:

    1-moveToElement (destino WebElement)
    2-moveToElement (destino WebElement, int xOffset, int yOffset)
